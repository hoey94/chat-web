链表是一种基本的数据结构，通过使用指针将一系列的节点串起来形成一个链表。在 Java 中，我们通常使用类来表示链表中的节点。

public class Node {\n    public int val; // 节点的值\n    public Node next; // 指向下一个节点的指针\n\n    public Node(int val) {\n        this.val = val;\n        this.next = null;\n    }\n}\n

节点中包含一个整数值 val，以及一个指向下一个节点的指针 next。构造函数用来初始化节点的值，以及将指针初始化为 null。

public class LinkedList {\n    private Node head; // 指向链表头部的指针\n    private int size; // 链表中节点的个数\n\n    public LinkedList() {\n        this.head = null;\n        this.size = 0;\n    }\n\n    // 在链表头部添加节点\n    public void addAtHead(int val) {\n        Node newNode = new Node(val);\n        newNode.next = head;\n        head = newNode;\n        size++;\n    }\n\n    // 在链表尾部添加节点\n    public void addAtTail(int val) {\n        Node newNode = new Node(val);\n        if (head == null) {\n            head = newNode;\n        } else {\n            Node curr = head;\n            while (curr.next != null) {\n                curr = curr.next;\n            }\n            curr.next = newNode;\n        }\n        size++;\n    }\n\n    // 在链表的第 index 个位置插入节点\n    public void addAtIndex(int index, int val) {\n        if (index < 0 || index > size) {\n            return;\n        }\n        if (index == 0) {\n            addAtHead(val);\n        } else if (index == size) {\n            addAtTail(val);\n        } else {\n            Node newNode = new Node(val);\n            Node curr = head;\n            for (int i = 0; i < index - 1; i++) {\n                curr = curr.next;\n            }\n            newNode.next = curr.next;\n            curr.next = newNode;\n            size++;\n        }\n    }\n\n    // 删除链表头部的节点\n    public void deleteAtHead() {\n        if (head != null) {\n            head = head.next;\n            size--;\n        }\n    }\n\n    // 删除链表尾部的节点\n    public void deleteAtTail() {\n        if (head == null) {\n            return;\n        }\n        if (head.next == null) {\n            head = null;\n        } else {\n            Node curr = head;\n            while (curr.next.next != null) {\n                curr = curr.next;\n            }\n            curr.next = null;\n        }\n        size--;\n    }\n\n    // 删除链表的第 index 个节点\n    public void deleteAtIndex(int index) {\n        if (index < 0 || index >= size) {\n            return;\n        }\n        if (index == 0) {\n            deleteAtHead();\n        } else if (index == size - 1) {\n            deleteAtTail();\n        } else {\n            Node curr = head;\n            for (int i = 0; i < index - 1; i++) {\n                curr = curr.next;\n            }\n            curr.next = curr.next.next;\n            size--;\n        }\n    }\n\n    // 获取链表中第 index 个节点的值\n    public int get(int index) {\n        if (index < 0 || index >= size) {\n            return -1;\n        }\n        Node curr = head;\n        for (int i = 0; i < index; i++) {\n            curr = curr.next;\n        }\n        return curr.val;\n    }\n\n    // 获取链表中节点的个数\n    public int size() {\n        return size;\n    }\n}\n
